# Nano-AI

**Nano NGU AI Assistant** is your intelligent assistant for ethical hacking, penetration testing, and cybersecurity guidance. Built with **Next.js + React**.

---

## Features

- AI Chat Assistant
- Voice Input Support
- Dark/Light Theme
- Secure & Private
- Responsive UI

---

## Installation

1. Clone the repository
```bash
git clone https://github.com/Alimoha-1/Nano-Ai.git
```
2. Navigate to the project folder
```bash
cd Nano-Ai
```

3. Install dependencies
```bash
npm install
```

4. Run the project
```bash
npm run dev
```
